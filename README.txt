== Description ==

World City Events: IXYT is a plugin for Wordpress that
makes it possible to insert live blocks with posters of various events in the cities you need around
the world into your website. It has a user-friendly interface and support for many languages.
This plugin does not collect any personal data.
More info about plugin usage on https://ixyt.info/en/plugin


World City Events: IXYT plugin is relying on a 3rd party as a service, the widget is using events data provided by ixyt.info.
* Service provider link – https://ixyt.info
* Service privacy policy and terms of use link – https://ixyt.info/en/privacy

== Installation ==

1. Upload the entire `world-city-events-ixyt` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the Plugins screen (Plugins > Installed Plugins).
3. Place `<?php do_shortcode('shortcode_given_by_plugin'); ?>` in your templates
